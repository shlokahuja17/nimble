# ⚡ Nimble - 2-Player Racing Game

A fast-paced, browser-based racing game where two players compete to avoid obstacles and collect coins!

## 🎮 Play Now

**Live Game:** [https://shlokahuja17.github.io/nimble/](https://shlokahuja17.github.io/nimble/)

## 📋 How to Play

### 2-Player Mode
- **Player 1**: Use `A` and `D` keys to move left and right
- **Player 2**: Use `← →` arrow keys to move left and right

### Challenge Mode
- Click **"Switch Role"** during the game to control both players with a single keyboard
- The highlighted player is the one you're currently controlling
- Switch between them to keep both alive!

### Objective
- Avoid the red blocks ❌
- Catch the gold squares 🪙 for bonus points
- The player with the most coins when the game ends wins!

## 🏆 Features

- **Real-time 2-Player Gameplay**: Control both players simultaneously
- **Dynamic Difficulty**: Game speed increases as you score higher
- **Leaderboard**: Track your best scores (stored locally in browser)
- **Particle Effects**: Visual feedback for coin collection
- **Responsive Design**: Works on desktop and tablet devices
- **Dark Theme**: Easy on the eyes with a modern gradient design

## 🛠️ Technical Details

- Built with vanilla HTML5, CSS3, and JavaScript
- Uses HTML5 Canvas for game rendering
- LocalStorage for leaderboard persistence
- No dependencies required
- Fully self-contained in a single HTML file

## 🚀 Getting Started

### Play Online
Simply visit: [https://shlokahuja17.github.io/nimble/](https://shlokahuja17.github.io/nimble/)

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/shlokahuja17/nimble.git
   ```
2. Open `index.html` in your web browser

## 💡 Tips

- Start slowly and get used to the controls
- In Challenge Mode, switch roles frequently to handle obstacles on both sides
- Collect bonus coins to increase your score faster
- Watch the difficulty ramp up - stay focused!

## 📊 Scoring System

- **Dodging obstacles**: +1 point
- **Collecting bonus coins**: +5 points
- **Coin calculation**: Points ÷ 50 = Final Coins (rounded)

## 🎨 Customization

Feel free to modify the game by editing `index.html`:
- Change colors in the `<style>` section
- Adjust game speed, difficulty, and spawn rates in the `<script>` section
- Modify canvas sizes and player dimensions
